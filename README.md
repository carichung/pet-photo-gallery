# Pet Photo Gallery

## Overview
This is a simple Pet Photo Gallery app deployed using Kubernetes and Terraform across multiple cloud providers.

## Setup
1. Clone this repository.
2. Install dependencies: `npm install` (for Node.js) or `pip install -r requirements.txt` (for Flask).
3. Run the app locally: `node app.js` or `flask run`.
4. Deploy to Kubernetes using Terraform: `terraform apply`.

## Requirements
- Docker
- Terraform
- Kubernetes cluster (AWS EKS, Azure AKS, or GCP GKE)
